Final Model Evaluation

I had originally made an ROC curve for this section, but the area under the curve was great enough to render the chart useless. Instead I have made these matrices which compare how changing the confidence threshold affects the true positive rate. The chart on the left assumes every prediction is correct regardless of the confidence. While the chart on the right only accepts predictions made with a probability above 95%. In this way I have turned a multi-class classification problem into a binary classification problem.

In this case, each classification is a representation of the following:
- True Positive: Correctly classified images with confidence above the threshold.
- True Negative: Incorrectly classified images with confidence below the threshold.
- False Positive: Incorrectly classified images with confidence above the threshold.
- False Negative: Correctly classified images with confidence below the threshold.

Due to the starting model being overfit on the training data, and this leading to me using the test data for model evaluation, there is a possibility that we will overfit to our test data. The best solution to this issue (and a great solution to almost any data science problem) would be get more data. While it was not possible for me to acquire thousands more photos for testing purposes, my sister agreed to take a few pictures for me. She lives in England, and while the signs there are not entirely the same, these photos provide a difficult and interesting final test.
